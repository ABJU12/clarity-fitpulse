# FitPulse
A decentralized fitness tracking and reward system built on Stacks blockchain.

## Features
- Create and manage workout profiles
- Track workout completion and achievements
- Earn FitPulse tokens for completing workouts
- Marketplace for workout plans and achievements

## Smart Contracts
- FitPulseCore: Main contract handling user profiles and workouts
- FitPulseToken: Reward token contract
- FitPulseMarket: Marketplace for workout plans
